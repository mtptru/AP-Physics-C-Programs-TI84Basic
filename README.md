# AP-Physics-C-Programs-TI84Basic
Unit by unit, miscellaneous, and full reference sheet for calculus-based college level physics intro courses  
Intended for use with TI-84Basic compatible calculators

# How to Use
___
1. Download [TI Connect CE](https://education.ti.com/en/products/computer-software/ti-connect-ce-sw)
2. Download desired programs (unit-by-unit, misc, and/or full reference sheet)
3. Connect to calculator using TI-84 proprietary min-5-pin USB cable
4. Upload directly to calculator's archive (recommended) or RAM
