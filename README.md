# AP-Physics-C-Programs-TI84Basic
Unit by unit, miscellaneous, and full reference sheet for calculus-based college level physics intro courses  
Intended for use with TI-84Basic compatible calculators

# How to Use
1. Download [TI Connect CE](https://education.ti.com/en/products/computer-software/ti-connect-ce-sw)
2. Download desired programs (unit-by-unit, misc, and/or full reference sheet)
3. Connect to calculator using TI-84 proprietary min-5-pin USB cable
4. Upload directly to calculator's archive (recommended) or RAM

# Programs
### Electricity and Magnetism
APEM(1-5) - unit by unit reference sheets corresponding to AP curriculum standards  
APEMCIRC - full circuits reference  
APEMMISC - miscellaneous physics E&M laws and formulas  
APEMREF - full course reference sheet aligned with AP curriculum  
EMCIRCS - more circuit reference with transient eq's  
EMTRANS - transients for circuits and other applications  

### Mechanics
APMECH[12,34,56] - unit by unit reference sheets (combined into pairs) corresponding to AP curriculum standards  
APMEFRQ - FRQ reference for common questions and problem solving routines  
APMEMISC - miscellaneous physics Mechanics laws and formulas  
APMEMIS2 - miscellaneous physics Mechanics units and theorems
